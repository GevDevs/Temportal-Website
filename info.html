Information

What is the goal of this site?

The goal of TEMPORTAL is to offer downloads of classic ROBLOX games that are compatible with their respected client. As a bonus, the site also offers a small selection of multiplayer games that can be played through current-day ROBLOX.

Is there a Discord server?

Yes. You can join the Discord server here.

How do I visit places in singleplayer?

To visit a place in singleplayer, you will need to download a client or client launcher to enjoy the game properly. Here are some options for clients.

  • Novetus (2006S to 2011M)
  • Roblox, March 2007 (Unpatched)
  • Roblox, August 2007 (Unpatched)
  • Roblox, Mid 2008 (Unpatched)
  • Roblox, Late 2009 (Unpatched)

If you want to play GodoBlocks games, they have to be opened in its dedicated client. You can download the client for Windows, MacOS and Linux with this link.

Requesting a Place

If you wish to add a place to TEMPORTAL, you can join the GodoBlocks & Temportal Discord Server and send a request in the 🗨place-requests forum channel. In the request, it is recommended to include,

  • A Roblox ID to an uncopylocked place (cannot be an illegal copy) or,
  • an RBXL file compatible with the client the place was developed for,
  • the name of the game,
  • the description of the game (No need to add one if there isn't one).

If you are unable to send a request in the Discord Server, you can also email blocksgodo@gmail.com.

Requesting a Game to Become Multiplayer

If you wish to request a place to become multiplayer on TEMPORTAL, you can join the GodoBlocks & Temportal Discord Server and send a request in the 🗨multiplayer-requests forum channel. In the request, it is recommended to include,

  • The link to the game on TEMPORTAL,
  • an explanation as to why the game would benefit from multiplayer.

You can also include a multiplayer explanation in 🗨place-requests if you're requesting a new place.

If you are unable to send a request in the Discord Server, you can also email blocksgodo@gmail.com.

What was GodoBlocks?

GodoBlocks was a 2005 Roblox revival created by myself (Gev) at the end of 2021. Because of my lack of experience in Godot at the time, the code grew messy and unoptimized. That along with me shifting my interests to making games based on my own original ideas meant GodoBlocks was unfortunately canceled in January of 2023.
